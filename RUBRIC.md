# Rubric

### Scoring (0–3 scale)
- 0 = Missing / Broken
- 1 = Poor
- 2 = Okay
- 3 = Good

## Functionality
- Runs in browser
- Slot machine works (spin + result)
- Token system works (win/spend)
- Uses AI humor/theme

## Code Quality
- Organized (HTML/CSS/JS structure)
- Readable
- Few bugs
- Not overly complicated

## Product Quality 
- UI looks decent
- Easy to use
- Feels complete
- Some creativity

### Additional Questions for consideration:
1. How it behaves, what works/doesn’t
2. Structure, bugs, anything notable
3. Anything surprisingly different?
